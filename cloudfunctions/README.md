# 类别管理功能说明

## 云数据库配置

### 1. 创建 categories 集合

在微信开发者工具中打开云开发控制台，**必须手动创建**集合 `categories`。

### 2. 添加类别数据

在 `categories` 集合中添加类别文档，每个文档包含以下字段：

```json
{
  "name": "类别名称",
  "createdAt": "2026-03-10T00:00:00.000Z",
  "updatedAt": "2026-03-10T00:00:00.000Z"
}
```

例如添加以下类别：
- 电子设备
- 房产
- 车辆
- 投资
- 其他

**注意**：如果数据库中没有类别数据，页面上的类别选择器将为空。

### 3. 数据库权限设置

`categories` 集合权限建议设置为：**所有用户可读，仅创建者可读写**

这样每个用户可以管理自己的类别，同时不影响其他用户。

## 云函数部署

部署以下云函数：

1. **getCategories**: 获取类别列表
2. **addCategory**: 添加新类别

在微信开发者工具中右键云函数文件夹 -> 上传并部署：云端安装依赖

## 功能说明

### 首页
- 顶部显示类别筛选栏，显示数据库中的所有类别
- 类别可以组合状态（服役中/已退役/已卖出）进行筛选

### 添加资产页
- 类别选择器显示数据库中的所有类别
- 点击 `+` 按钮可以添加新类别到云数据库
- 新添加的类别会立即生效

## 注意事项

1. **必须先创建 categories 集合并添加数据**，否则类别选择器为空
2. 如果云函数调用失败，页面会显示错误提示
3. 用户可以在添加资产页面添加新类别，会自动保存到云数据库
