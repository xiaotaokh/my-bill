# 配色方案 - 商务黑白灰风格

## 设计原则

1. **稳重专业** - 深灰为主色调，避免纯黑的激进感
2. **黑白灰基调** - 中性色调，不引入彩色干扰
3. **层级克制** - 通过灰度深浅区分，变化柔和

---

## 核心配色

### Neutral 中性色（主色调）

| 名称 | HEX | 用途 |
|------|-----|------|
| Dark | `#333333` | 主按钮背景、重要信息 |
| Gray-900 | `#1A1A1A` | 主文字、标题 |
| Gray-800 | `#2D2D2D` | 重要文字 |
| Gray-700 | `#555555` | 次级文字、说明 |
| Gray-500 | `#888888` | 占位文字、禁用状态 |
| Gray-400 | `#AAAAAA` | 辅助元素 |
| Gray-300 | `#CCCCCC` | 边框、分割线 |
| Gray-200 | `#E5E5E5` | 输入框背景、区块背景 |
| Gray-100 | `#F5F5F5` | 页面背景 |
| White | `#FFFFFF` | 卡片背景、按钮文字 |

---

### Status 状态色（仅用于提示）

| 名称 | HEX | 用途 |
|------|-----|------|
| Success | `#52C41A` | 成功状态、服役中 |
| Error | `#CF1322` | 错误状态、删除操作 |
| Warning | `#D48806` | 警告状态、待生效 |

---

## 使用场景

| 元素 | 颜色 |
|------|------|
| 主按钮背景 | Dark `#333333` |
| 主按钮文字 | White `#FFFFFF` |
| 边框按钮 | White 背景 + Dark 边框 `#333333` |
| 卡片背景 | White `#FFFFFF` |
| 页面背景 | Gray-100 `#F5F5F5` |
| 输入框背景 | Gray-200 `#E5E5E5` |
| 主文字 | Gray-900 `#1A1A1A` |
| 次级文字 | Gray-700 `#555555` |
| 占位文字 | Gray-500 `#888888` |
| 边框分割线 | Gray-300 `#CCCCCC` |

```css
  --gray-500: #888888;
  --gray-400: #AAAAAA;
  --gray-300: #CCCCCC;
  --gray-200: #E5E5E5;
  --gray-100: #F5F5F5;
  --white: #FFFFFF;

  /* Status - 状态色 */
  --success: #52C41A;
  --error: #CF1322;
  --warning: #D48806;
}
```

## 商务风格要点

- **主按钮**：深灰 `#333333` 背景，稳重不激进
- **文字层级**：柔和过渡 `#1A1A1A` → `#555555` → `#888888`
- **背景层级**：浅灰渐进 `#FFFFFF` → `#F5F5F5` → `#E5E5E5`
- **状态色**：偏暗偏稳，不使用高饱和亮色
- **整体氛围**：克制、稳重、专业
