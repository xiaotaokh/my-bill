# 微信小程序 - 朝夕数计 (myBill)

> 个人资产管理工具，帮助你统计和追踪名下各类资产

**版本**: v1.0.0 | **开发时间**: 2026年3月10日

## 项目说明

这是一个微信小程序项目，用于统计和管理个人名下资产。支持：
- 添加和管理个人资产
- 统计资产总览（总价、日均成本等）
- 按状态分类（服役中/已退役/已卖出）
- 多维度筛选和排序
- 数据云端同步，多设备访问

## 快速开始（5分钟上手）

### 第1步：导入项目
1. 打开微信开发者工具
2. 点击「导入项目」
3. 选择本项目目录
4. AppID 已配置：`wxb1fb63721cb2da59`

### 第2步：开通云开发
1. 点击顶部菜单「云开发」
2. 点击「开通」
3. 选择付费方式（可设置免费资源包）
4. **记下环境ID**（格式：production-xxx）

### 第3步：配置环境

**方法一（推荐）**：在开发者工具中配置
- 点击右上角「详情」→「本地设置」
- 在「云开发环境」中选择你的环境

**方法二**：修改代码
- 打开 `app.js`
- 修改第6行的 `env: 'production-xxx'` 为你的环境ID

### 第4步：初始化数据库

1. 在云开发控制台点击「数据库」
2. 点击「+」创建集合，名称：`assets`
3. 权限设置：「仅创建者可读写」

### 第5步：部署云函数

1. 展开 `cloudfunctions` 目录
2. 右键点击每个云函数文件夹
3. 选择「上传并部署：云端安装依赖」
4. 等待部署完成

**需要部署的云函数**：
- `addAsset` - 添加资产
- `getAssets` - 获取资产列表
- `getStats` - 获取统计信息
- `updateAsset` - 更新资产
- `deleteAsset` - 删除资产

### 第6步：运行项目

- 点击工具栏的「编译」按钮
- 在模拟器中查看效果

## 功能说明

### 1. 首页
- 显示资产总览统计（总价、日均成本、各状态数量）
- 支持按状态筛选（全部/服役中/已退役/已卖出）
- 支持按多个维度排序（价格、购买时间、服役时长、日均成本）
- 点击资产卡片查看详情
- 下拉刷新功能
- 浮动添加按钮

### 2. 添加资产
- 填写物品名称、价格、购买日期
- 选择或添加类别（支持自定义）
- 填写备注（可选）
- 设置状态（服役中/已退役/已卖出）
- 设置是否不计入总资产/日均成本
- 表单验证

### 3. 资产详情
- 查看资产详细信息
- 显示服役时长和日均成本计算
- 编辑资产
- 删除资产（带确认提示）

## 数据字段说明

### 状态定义

| 状态 | 值 | 说明 |
|------|-----|------|
| 服役中 | `active` | 正在使用 |
| 已退役 | `retired` | 不再使用 |
| 已卖出 | `sold` | 已出售 |

### 资产分类

默认分类：电子设备、房产、车辆、投资、其他（支持自定义添加）

### 资产表（assets）

| 字段 | 类型 | 说明 |
|------|------|------|
| _id | String | 唯一标识 |
| _openid | String | 用户openid |
| name | String | 资产名称 |
| category | String | 资产分类 |
| price | Number | 价格 |
| purchaseDate | Date | 购买日期 |
| status | String | 状态（active/retired/sold） |
| remark | String | 备注 |
| excludeTotal | Boolean | 不计入总资产 |
| excludeDaily | Boolean | 不计入日均 |
| createdAt | Timestamp | 创建时间 |
| updatedAt | Timestamp | 更新时间 |

### 计算字段

- **服役时长**: 当前日期 - 购买日期（仅状态为"服役中"时计算）
- **日均成本**: 价格 / 服役天数

## 常见问题

### Q: 云函数部署失败？
A: 检查网络连接，确保已开通云开发，重新部署

### Q: 数据库操作失败？
A: 检查数据库权限是否设置为「仅创建者可读写」

### Q: 如何重置数据？
A: 在云开发控制台的数据库中删除 `assets` 集合中的数据

### Q: 如何添加新的类别？
A: 在添加资产页面，点击类别旁边的「+」按钮即可添加新类别

### Q: 数据是否支持多设备同步？
A: 是的，所有数据存储在云端，登录同一微信账号可在多设备访问

### Q: 环境ID在哪里查看？
A: 在云开发控制台的右上角可以看到当前环境的ID
