# 📱 微信小程序 - 我的资产 (myBill)

> 个人资产管理工具，帮助你统计和追踪名下各类资产

---

## 🎯 项目概述

这是一个基于微信原生云开发的资产管理小程序，支持：
- ✅ 添加和管理个人资产
- ✅ 统计资产总览（总价、日均成本等）
- ✅ 按状态分类（服役中/已退役/已卖出）
- ✅ 多维度筛选和排序
- ✅ 数据云端同步，多设备访问

## 🚀 快速开始（3步搞定）

### 第1步：导入项目
1. 打开微信开发者工具
2. 点击「导入项目」
3. 选择本项目目录

### 第2步：开通云开发
1. 点击顶部「云开发」
2. 点击「开通」
3. 记下环境ID

### 第3步：初始化
1. 创建数据库集合 `assets`
2. 部署所有云函数
3. 点击「编译」运行

**详细步骤请参考：[快速开始.md](./快速开始.md)**

## 📊 数据模型

```javascript
{
  "_id": "唯一标识",
  "_openid": "用户openid",
  "name": "资产名称",
  "category": "资产分类",
  "price": 10000,              // 价格
  "purchaseDate": "2024-01-01", // 购买日期
  "status": "active",           // 状态
  "remark": "备注信息",
  "excludeTotal": false,        // 不计入总资产
  "excludeDaily": false,        // 不计入日均
  "createdAt": timestamp,       // 创建时间
  "updatedAt": timestamp        // 更新时间
}
```

## 📝 使用说明

### 添加资产
1. 点击首页右下角「+」按钮
2. 填写资产信息
3. 选择或添加类别
4. 设置状态和特殊选项
5. 点击「保存资产」

### 查看统计
- 首页自动显示资产总览
- 顶部卡片展示关键数据
- 支持实时更新

### 筛选排序
- 点击状态标签筛选
- 点击排序下拉框选择排序方式
- 实时刷新列表

## 🔧 配置说明

### 必要配置
1. **云开发环境**: 需要开通云开发并配置环境ID
2. **数据库**: 创建 `assets` 集合，权限设为「仅创建者可读写」
3. **云函数**: 部署所有5个云函数

### 可选配置
- 修改 `app.json` 中的页面标题
- 修改 `app.wxss` 中的全局样式
- 添加更多资产类别

## 🐛 常见问题

### Q: 云函数部署失败？
A: 检查网络连接，确保已开通云开发，重新部署

### Q: 数据库操作失败？
A: 检查数据库权限是否设置正确

### Q: 如何添加测试数据？
A: 参考 `test-data.js`，在控制台运行 `addTestData()` 函数

### Q: 数据支持多设备同步吗？
A: 是的，数据存储在云端，登录同一微信账号即可同步
