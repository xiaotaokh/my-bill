# 朝夕数计365小程序功能简介与使用手册

## 一、功能简介

朝夕数计365是一款个人资产管理微信小程序，用于记录固定资产和订阅类资产，帮助用户持续了解资产总投入、日均成本、使用周期、分类分布和购买趋势。

小程序当前已支持以下核心能力：

- 资产总览：展示计入总资产、计入总日均成本、资产状态数量、当前筛选下的总额和日均成本。
- 资产记录：支持新增、编辑、删除固定资产和订阅资产。
- 订阅管理：支持月度、季度、年度、周、自定义周期，以及待生效、订阅中、已结束状态。
- 成本分析：自动计算服役天数、订阅天数、总投入、日均成本和折合每日成本。
- 筛选排序：支持按状态、分类、关键词筛选，并按价格、购买时间、添加时间、服役时长、日均成本排序。
- 统计报表：支持资产概览、分类分布、购买趋势、时段分布等图表分析。
- 分类管理：支持添加、编辑、删除、批量管理资产分类，可使用内置图标、自定义emoji或上传图片作为分类图标。
- 个性化设置：支持头像昵称维护、主题切换、金额显隐、列表/简洁视图切换。
- 天气与日期：首页展示当前日期和本地天气信息。
- 管理员统计：管理员可查看用户总览、访问趋势和用户资产明细。

## 二、页面说明

### 1. 首页

首页是资产管理的主入口，包含日期天气、用户信息、核心统计、状态筛选、分类筛选、资产列表和快捷操作。

主要区域说明：

- 顶部日期与天气：展示当天日期、星期和定位天气。
- 用户头像昵称：点击可进入账户页面。
- 计入总资产：统计未勾选“不计入总资产”的资产金额。
- 计入总日均成本：统计未勾选“不计入总日均”的日均成本。
- 已排除提示：展示被排除在总资产或总日均之外的资产数量。
- 状态卡片：可按“服役中/订阅中”“待生效”“已退役/已结束”“已卖出”筛选。
- 分类标签：可按资产分类筛选列表。
- 资产列表：展示资产名称、类型、状态、分类、日期范围、金额、日均成本等信息。
- 搜索按钮：按资产名称搜索。
- 添加按钮：进入新增资产页面。
- 设置按钮：进入设置面板。

### 2. 新增/编辑资产页

该页面用于录入固定资产或订阅资产。顶部可选择资产类型：

- 固定资产：适合一次性购买的物品，例如手机、电脑、家具、车辆等。
- 订阅资产：适合周期性付费项目，例如会员、软件订阅、云服务、AI工具订阅等。

固定资产可填写：

- 物品名称
- 价格
- 购买日期
- 图标或图片
- 分类
- 备注
- 状态：服役中、已退役、已卖出
- 退役日期或卖出日期
- 是否不计入总资产
- 是否不计入总日均

订阅资产可填写：

- 订阅名称
- 每期金额
- 周期类型：月度、季度、年度、周、自定义
- 周期计算方式：自然月/季/年、日对日、日对日减一
- 自定义周期天数：1-365天
- 订阅日期或订阅开始日期
- 是否待生效
- 是否结束订阅
- 订阅结束日期
- 是否不计入总资产
- 是否不计入总日均
- 图标、分类和备注

保存时会进行表单校验，例如名称必填、价格或每期金额必须有效、分类必选、结束状态需要填写对应日期等。

### 3. 资产详情页

资产详情页用于查看单个资产的完整信息。

可查看内容包括：

- 资产名称、图标、分类、备注
- 固定资产价格或订阅资产每期金额
- 购买日期、订阅日期、退役日期、卖出日期、结束日期
- 服役时长、订阅时长、日期范围
- 日均成本、折合每日成本、总投入
- 是否计入总资产和总日均

可执行操作：

- 编辑资产：进入编辑页面修改信息。
- 删除资产：删除前会弹出确认提示；如果资产使用了上传图片，会尝试同步清理存储文件。
- 预览图片：资产图标为上传图片时可预览原图。

### 4. 统计报表

从首页底部快捷操作进入统计报表。报表用于查看资产的整体结构和变化趋势。

当前报表包括：

- 总览统计：展示全量、计入、排除等不同口径下的总资产与日均成本。
- 状态统计：按服役中/订阅中、待生效、已退役/已结束、已卖出汇总数量和金额。
- 分类分布：按分类统计资产金额、数量和占比，并支持点击查看该分类下的资产列表。
- 购买趋势：按时间展示资产投入趋势，并支持查看某个时间点对应的资产。
- 时段分布：支持按全部、近一年、近半年、近三个月等时段查看资产分布；在“全部”视图下可切换时间粒度。

口径说明：

- 分类分布用于查看全部资产结构，包含“不计入总资产”的资产。
- 首页顶部“计入总资产”和“计入总日均成本”只统计未排除的资产。
- 订阅资产金额优先使用累计总投入，而不是简单使用每期金额。
- 待生效订阅可计入资产数量，但在未开始前不贡献日均成本。
- 已结束订阅不再计入当前日均成本。

### 5. 分类管理

分类管理用于维护资产分类，入口在首页设置面板中。

支持功能：

- 添加分类：填写分类名称、描述，选择图标。
- 编辑分类：修改名称、描述和图标。
- 查看详情：查看分类名称、图标、描述和资产数量。
- 删除分类：分类下有资产时不能删除。
- 批量管理：可批量选择并删除没有资产的分类。
- 排序：支持按创建时间排序，也支持自定义排序；自定义排序会影响首页分类显示顺序。

分类图标支持三种方式：

- 内置emoji图标
- 自定义emoji
- 上传图片，常见图片格式，大小不超过5M

### 6. 账户页面

账户页面用于维护当前用户展示信息。

支持功能：

- 查看当前头像、昵称和用户身份。
- 修改头像：可选择微信头像、相册头像或随机头像。
- 修改昵称：可使用微信昵称输入能力，也可使用随机昵称。
- 预览头像：上传或微信头像支持预览，内置随机头像不支持预览。

头像和昵称仅用于小程序内展示，不影响微信账号本身。

### 7. 主题设置

主题入口在首页设置面板中。当前内置主题包括：

- 星辰靛蓝
- 大地暖息
- 黑曜夜境
- 森林呼吸
- 日落暖橙
- 极光幻彩

切换主题后，小程序会保存选择，并同步应用到首页、资产页、分类页、账户页和统计页。

### 8. 管理员用户统计

管理员账号可在设置面板中进入用户统计页。

支持查看：

- 用户总数
- 资产总数
- 活跃用户
- 今日访问和昨日访问
- 访问趋势图
- 用户列表、用户资产数量、资产总额、最近访问时间
- 展开查看用户资产列表
- 删除用户及其资产、分类和图片数据

该功能仅管理员可见。

## 三、常用操作流程

### 1. 添加一个固定资产

1. 在首页点击右下角添加按钮。
2. 选择“固定资产”。
3. 填写名称、价格和购买日期。
4. 选择资产图标和分类。
5. 根据需要填写备注。
6. 选择资产状态。
7. 如不希望影响首页统计，可勾选“不计入总资产”或“不计入总日均”。
8. 点击“保存资产”。

### 2. 添加一个订阅资产

1. 在首页点击右下角添加按钮。
2. 选择“订阅资产”。
3. 填写订阅名称和每期金额。
4. 选择周期类型，例如月度、季度、年度、周或自定义。
5. 月度、季度、年度订阅可选择周期计算方式。
6. 如果订阅还未开始，开启“待生效”并填写开始日期。
7. 如果订阅已经结束，开启“结束订阅”并填写结束日期。
8. 选择图标、分类并保存。

### 3. 查找资产

可使用以下方式查找：

- 点击搜索按钮，输入资产名称搜索。
- 点击状态卡片，按状态筛选。
- 点击分类标签，按分类筛选。
- 使用排序选择器，按价格、购买时间、添加时间、服役时长或日均成本排序。
- 点击视图切换按钮，在详细视图和简洁视图之间切换。

### 4. 编辑或删除资产

1. 在首页点击资产卡片进入详情页。
2. 点击编辑进入编辑页面，修改后保存。
3. 如需删除，在详情页点击删除，并在确认弹窗中确认。

### 5. 批量删除资产

1. 从首页进入设置面板。
2. 进入批量删除资产功能。
3. 可搜索资产名称或分类。
4. 勾选需要删除的资产。
5. 点击删除并确认。

### 6. 管理分类

1. 从首页进入设置面板。
2. 点击分类管理。
3. 可添加新分类、编辑已有分类、调整排序或批量删除空分类。
4. 如果分类下已有资产，需要先修改或删除相关资产后才能删除分类。

### 7. 查看统计分析

1. 从首页底部快捷操作进入统计报表。
2. 查看总览、状态统计、分类分布、购买趋势和时段分布。
3. 点击图表或列表项，可查看对应分类、时间点或时段下的资产明细。

### 8. 隐藏金额

在首页“计入总资产”标题右侧点击眼睛图标，可切换金额显示和隐藏。隐藏后总资产金额会以星号展示。

### 9. 切换主题

1. 从首页进入设置面板。
2. 点击主题设置。
3. 选择喜欢的主题。
4. 主题会立即生效并保存。

## 四、统计字段解释

| 字段 | 含义 |
| --- | --- |
| 计入总资产 | 未被标记为“不计入总资产”的资产金额合计 |
| 计入总日均成本 | 未被标记为“不计入总日均”的资产日均成本合计 |
| 总投入 | 订阅资产根据周期和已完成期数计算出的累计投入 |
| 日均成本 | 当前仍在使用或订阅中的资产按已使用天数折算的每日成本 |
| 折合每日 | 用于展示单个资产或订阅的平均每日花费参考值 |
| 服役中/订阅中 | 固定资产处于服役中，或订阅资产处于订阅中 |
| 待生效 | 订阅已记录但开始日期未到或设置为待生效 |
| 已退役/已结束 | 固定资产已退役，或订阅资产已结束 |
| 已卖出 | 固定资产已出售 |
| 不计入总资产 | 该资产不参与首页计入总资产统计 |
| 不计入总日均 | 该资产不参与首页计入总日均成本统计 |

## 五、数据与权限说明

- 小程序通过微信登录获取用户openid，用于区分不同用户的数据。
- 资产、分类、用户信息存储在Supabase。
- 上传的头像、资产图标和分类图标存储在Supabase Storage。
- 同一微信用户在不同设备访问时，可读取同一份云端数据。
- 位置权限仅用于获取本地天气；拒绝授权不影响资产管理核心功能。

## 六、使用建议

- 固定资产建议填写真实购买日期，方便计算服役时长和日均成本。
- 订阅资产建议按实际扣费周期填写每期金额和周期类型。
- 对不希望影响主统计的数据，例如临时记录、报销项、赠品或历史参考项，可使用“不计入总资产”或“不计入总日均”。
- 分类建议保持稳定，分类名称变更会影响资产归类展示。
- 上传图片建议控制在5M以内，以保证加载速度和保存成功率。
